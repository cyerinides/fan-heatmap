Meta Ads Manager Optional

Export a geographic breakdown from your campaigns.

Recommended export settings:
  1. Set date range to Last 90 days
  2. Select the campaigns you want to include
  3. Breakdown → By Delivery → Region (for US state-level detail) or Country (for international)
  4. Export → Export Table Data → .csv

Shopify Optional

Recommended export settings:
